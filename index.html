Advanced Programming Project: Video Game Library

The Video Game Library is a complete two-tier application built using a Client-Server architecture. The **Backend (Server)** is implemented with **Flask**, providing six RESTful HTTP endpoints for managing media data, and ensuring data persistence by storing records in a JSON file. The **Frontend (Client)** is built with **Tkinter** and communicates exclusively with the server via HTTP requests to retrieve, display, create, and delete media items. This project demonstrates modular programming, robust pathing logic, and clean separation of concerns.
